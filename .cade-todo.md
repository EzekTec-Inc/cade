# Command Palette Implementation Plan

## Architecture
- **State**: `CommandPaletteState` in `app/mod.rs` (query, filtered_commands, cursor)
- **Registry**: Reuse SECTIONS from `menu.rs` (60+ commands already catalogued)
- **Fuzzy match**: Simple substring + scoring (no new deps)
- **Keybinding**: `Ctrl+P` opens, Esc closes, Enter executes, Up/Down navigate, type to filter
- **Rendering**: Centered overlay using existing `overlay.rs` helpers
- **Action**: Returns selected command string as if user typed it

## Files to create/modify:
1. `crates/cade-tui/src/app/command_palette.rs` (NEW) — state + fuzzy logic + tests
