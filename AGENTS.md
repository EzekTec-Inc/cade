# Agent Configuration and Instructions

## Agent skills

### Issue tracker

Issues and tasks are tracked using GitHub Issues (via the `gh` CLI). External Pull Requests are not included as a request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage is managed using the canonical defaults (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

The repository is configured as a single-context project. See `docs/agents/domain.md`.
